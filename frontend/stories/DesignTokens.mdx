# Design Tokens

Finchippay extends Tailwind CSS with a Stellar blue scale, deep Cosmos surfaces, semantic theme variables, and three purpose-specific font families. Use the named tokens instead of introducing one-off values.

## Spacing

The interface follows Tailwind's quarter-rem base scale. These are the recurring layout tokens used by cards, forms, and page shells.
